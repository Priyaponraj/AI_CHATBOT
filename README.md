AI Chatbot Web Application

This is a web-based AI chatbot built using Flask and OpenRouter API.
It allows users to chat with an AI model through a simple interface.

Features
- Real-time chat
- Simple UI
- AI-generated responses
- Continuous conversation

Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- OpenRouter API

Project Structure
chatbot/
├──AI_CHATBOT.py
├── app.py
├── templates/
   └── index.html

Installation
1. Install dependencies
pip install flask openai

2. Add your API key
Open app.py and AI_CHATBOT.py add your API key:
api_key="your_api_key"

4. Run the project
python app.py

5. Open in browser
http://127.0.0.1:5000/
